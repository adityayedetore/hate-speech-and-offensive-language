## How to run our code

Requirements
* python 3.7.0
* pytorch 1.7.0
* torchtext 0.8.0
* A `cuda` enabled device

To train the models, run `python3.7 main.py`. 

To generate the plots, and view some other statistics there is a ipython notebook `analysis.ipynb`, which requires R to run.
